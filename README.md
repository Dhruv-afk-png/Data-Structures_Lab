## Problem Statement
Write a C program to check whether a given element is present in an array of elements using linear search. The program should prompt the user to enter the size of the array, the elements of the array, and the element to search for.

---

## Algorithm
1. Start.  
2. Read the marks obtained by the student.
3. If marks > 40
   → Display “Pass”.
4. Else
   → Display “Fail”.
5. Stop.
